# Default Flowchart Legend

## Shapes
- Terminator: Start / End
- Process: Task or computation step
- Decision: Condition branch
- Data/Input: Incoming payload
- Data Store: Persistent artifact
- External Entity: Hardware/system boundary

## Lines
- Solid arrow: active implemented flow
- Dashed arrow: planned/not active flow
- Loopback: retry/regeneration path

## Colors
- Blue: process/terminator
- Amber: decision
- Green: data/data store
- Gray: external entities
- Muted gray dashed: planned branch
